# Исследование конкурентоспособности моделей прогнозирования на разных участках горизонта

Данный репозиторий содержит исследовательский проект в рамках домашнего задания по анализу временных рядов. 
Основная цель проекта - проверить гипотезу о том, что разные модели и стратегии (в частности, Recursive и Direct для CatBoost) выигрывают на разных шагах прогнозного горизонта.

Эксперименты проводятся на случайной выборке из 150 ежемесячных рядов датасета M4. Сравниваются статистические бейзлайны (StatsForecast), глобальный градиентный бустинг (CatBoost) и нейросеть (PatchTST).

## Воспроизводимость экспериментов

Для запуска проекта выполните следующие шаги:

**1. Подготовка окружения**

Убедитесь, что у вас установлен Python 3.9+. Установите необходимые зависимости из корня проекта:

`pip install -r requirements.txt`

**2. Запуск пайплайна**

Обучение всех моделей (StatsForecast, CatBoost Recursive/Direct, NeuralForecast) и формирование итогового файла с предсказаниями выполняется одним скриптом. Находясь в корневой папке, выполните:

`python run_experiment.py`

*Примечание: скрипт автоматически загрузит данные из `data/sampled_m4_150.csv` и сохранит результаты в папку `results/`.*

**3. Анализ результатов**

После успешного завершения эксперимента откройте ноутбук `analysis_results.ipynb`. Выполните ячейки, чтобы получить сводную таблицу глобальных метрик и графики распределения ошибки в разрезе шагов горизонта прогнозирования.
